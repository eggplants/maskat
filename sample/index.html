■ サンプルアプリケーション一覧

  • 四則演算

    • ユーザが入力した 2 つの値を用いて、足し算、引き算、掛け算、割り算を行う簡単なアプリケーションです。マスカットアプリケーションの基本的な動作や開発方法について理解するための入門的なサンプルとなっています。

  • ペットストア

    • Java Pet Store をモチーフに、マスカットを用いて Ajax リッチクライアント化したものです。従来の Java Pet Store ではフォームデータの送信ごとに画面全体が再描画され、紙芝居のように動作のたびに画面が切り替わっていました。マスカット Pet Store では画面内で更新が必要な部分だけが再描画され、通常のウィンドウアプリケーションのような操作感が実現されています。

  • フルーツショップ

    • 顧客からの受注を登録する業務アプリケーションのサンプルです。このサンプルではユーザからの入力をリアルタイムで検証し、エラーメッセージを表示する機能が使用されています。このサンプルでは LiveValidation プラグインを使用しています。

  • レストラン検索

    • Google Maps と連携したアプリケーションのサンプルです。銀座周辺のレストラン (データは架空のものです) を検索し、地図上に表示します。このサンプルでは Google プラグインを使用しています。
